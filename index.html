Caesar Cipher (Classical Cryptography)



    
    

        
How the Caesar Cipher Works

        

            The Caesar Cipher is one of the oldest known encryption techniques,
            used by Julius Caesar to protect military messages. It works by
            shifting each letter in the alphabet forward or backward by a
            fixed number of positions.
        

        
    
            
  • ✅ A simple substitution cipher
    • 
            
  • ✅ Symmetric (same key for encryption + decryption)
    • 
            
  • ✅ Easy to break using brute force (only 25 shifts!)
    • 
        


        
        

    







    
SHA‑256 Hash (One‑Way Hashing)


    
    

    


    
    

        
Understanding SHA‑256

        

            SHA‑256 is a cryptographic hash function that transforms input into
            a 256‑bit value. It is one‑way, meaning the original
            message cannot be recovered from the hash.
        

        
    
            
  • ✅ Deterministic (same input → same output)
    • 
            
  • ✅ Huge output space (2256)
    • 
            
  • ✅ Used in blockchain (Bitcoin), digital signatures, password storage
    • 
        

    







    
AES‑256 Encryption (Modern Symmetric Crypto)


    
    

    
    

    


    
    

        
How AES‑256 Works

        

            AES is the global standard for modern encryption and secures everything
            from HTTPS connections to Wi‑Fi and hard drives. AES‑256 uses a 
            256‑bit key and 14 rounds of transformation.
        

        
    
            
  • ✅ Symmetric block cipher
    • 
            
  • ✅ Extremely secure (2256 possible keys)
    • 
            
  • ✅ Resistant to modern cryptanalysis
    •
